üìå Why this works


Each table maintains its own independent AUTO_INCREMENT counter.


When you run:


INSERT INTO personal_customer_sequence_table VALUES ();

MySQL inserts a row with a new auto-increment value.


Calling:


SELECT LAST_INSERT_ID();

returns the generated sequence number for your connection.

You can make the code more reusable:
private Long getNextSeqId(String tableName) {
    jdbc.update("INSERT INTO " + tableName + " VALUES ()");
    return jdbc.queryForObject("SELECT LAST_INSERT_ID()", Long.class);
}

Then call:
public Long getNextCustomerSeqId() {
    return getNextSeqId("personal_customer_sequence_table");
}
